GUI library
===========

Event types
------------

 * Generic mouse events:
    - ``mouseMovedTo(Math::Vector)``
    - ``mouseEntered()``
    - ``mouseLeft()``
    - ``mouseButtonDown(int)``
    - ``mouseButtonUp(int)``
    - ``mouseClicked(int)``

 * ``LineEdit``
    - ``gainedFocus()``
    - ``lostFocus()``
    - ``textChanged()``
